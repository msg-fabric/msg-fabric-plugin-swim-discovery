# Docker Example for msg-fabric-plugin-swim-discovery

Start local docker registry and demo network —

```bash
% npm -s run start
```

Start Swim Core —

```bash
% cd swim-core
% npm -s run start
```

Start Service One —

```bash
% cd service-one
% npm -s run start
```

Start Service Two —

```bash
% cd service-two
% npm -s run start
```
